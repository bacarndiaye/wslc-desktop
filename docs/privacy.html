Privacy Policy

Effective date: July 6, 2026 · Applies to the WSLC Desktop application and this website

In one sentence WSLC Desktop collects no user data — no telemetry, no analytics, no accounts, no crash reporting — and does not transfer any information to other networked systems.

What the application does

WSLC Desktop is a local, open-source desktop application for managing WSL containers. Everything it displays comes from invoking the wslc.exe command-line tool already installed on your computer. All processing happens on your machine.

Data collection

  • Personal data: none is collected, stored, or shared.
  • Telemetry & analytics: none. The app contains no tracking, metrics, or crash-reporting code.
  • Accounts: none. The app has no sign-in and no cloud component.

Local data

The app stores its own preferences (theme, refresh interval, similar settings) in a JSON file inside your Windows user profile (the standard Electron userData folder). This file never leaves your computer and is removed if you delete that folder; uninstalling keeps it unless you remove it yourself.

Network activity

The application itself makes no network requests. The only network activity is what you explicitly request through wslc — for example pulling a container image — and it is performed by wslc.exe, not by WSLC Desktop. The app has no auto-update mechanism; updates are downloaded manually from GitHub Releases.

This website

This site is hosted on GitHub Pages and contains no analytics or cookies of its own. GitHub may collect visitor logs as described in the GitHub Privacy Statement.

Changes

Any change to this policy is published on this page and recorded in the project's public commit history.
