The session bar

wslc keeps a separate world per user and elevation level. The bar always shows which session you're in, flags Administrator sessions, and turns preview errors into plain guidance.

Install in a minute

  1. Make sure the WSL container preview is installed — wslc list must work in a terminal.
  2. Download WSLC Desktop Setup from the latest release and run it (per-user, no admin rights).
  3. Launch WSLC Desktop. Your containers, images, volumes and networks are already there.
Tip — stay non-elevated. wslc keeps one session per elevation level: an elevated app sees different containers than your normal terminals. Run WSLC Desktop (and your terminals) without administrator rights; the session bar confirms where you are.
